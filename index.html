HTML Elements

<html>

Represents the root level of an HTML document. All the other elements should be child of this element.

<head>

It contains machine readable information, metadata, about the document such as title, scripts and style sheets

<link>

It explains the relationship between current document and external resource, Mostly used in linking CSS files

<style>

It contains CSS which is applied to element

<body>

It represents HTML document, it is the visible part on webpage

<header>

It represents top section of website , usually contains logo, menus , search form

<footer>

It represents bottom section of website, usually contains information about copyright data, author information or other links

<h1>,
<h2>,
<h3>,
<h4>,
<h5>

They represent heading levels of each section, h1 is highest and h5 is lowest

<main>

It represents the main section of document, that contain topics and sections that are directly related to

<nav>

It represents section of page that is dedicated to providing link within the documents or external sources

<div>

It is a generic container for content, and has no effect on layout untill styled by CSS

<li>

It represents an item in the list. Its must have a parent element <ul>, <ol> or <menu>

<a>

It creates a hyperlink to webpages, files, location within document or outside

<br>

It is used for line break in text, usually used while writing a poem or address

<audio>

It is used to embed sound in document.

<img>

It is used for embedding image into a document

<video>

It is used for embedding media player that supports video playback

<script>

It is used to embed executable code, usually used for adding javascript code to document

<table>

It represents data in tabular format in two dimensions, row and column

<form>

It represent a section of document that has interactive elements for submitting an information

<input>

It is used to create interactive controls to accept data from user

<label>

It represent a caption for an element

<select>,<datalist>

It represent a control, that has menu of options

<option>

It represents an item within select or datalist

<progress>

It represent a progress bar

<textarea>

It represents a multi-line plain text area, usually comment or a feedback area.

<details>

It is used for creating a disclosure widget, using summary or label element (kind of like tap to expand)
